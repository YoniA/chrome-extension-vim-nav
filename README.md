

### Supported commands

| Key  | Functionality            | Notes    |
| ---- | ------------------------ | --- |
| `j`  | Scroll down              |     |
| `k`  | Scroll up                |     |
| `l`  | Scroll right             |     |
| `h`  | Scroll left              |     |
| `gg` | scroll to Top of page    |     |
| `G`  | scroll to bottom of page |     |
| `H`  | Back (history)           |     |
| `L`  | Next (history)           |     |
